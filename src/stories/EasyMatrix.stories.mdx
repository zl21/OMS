## EasyMatrix

### 一、组件说明
#### 1. 概述
 简单矩阵，其中只是封装了`businessForm`组件。

#### 2. 应用场景
比如添加赠品弹窗，二维的表单，私仓业务组件库中，需三维，故此组件尚未在业务组件库中使用。

### 二、使用说明

> 应用

```js
<EasyMatrix :componentData="componentData" />
```

```js
export default {
  components: {
    easyMatrix
  },
  data() {
    return {
      componentData: {
        fun: obj => {
          console.log(obj~~~~)
        }
      }
    }
  }
}
```
