# 国际化

```js
// index.html
<script src="https://cdn.jsdelivr.net/npm/@bojun/i18n@1.2.4/dist/index.min.js"></script>
```

```js
// webpack.config.js 没有import则不用设置
externals: {
	'@burgeon/internationalization/i18n': '$i18n',
  '@bojun/i18n': '$i18n'
}
```

```js
// main.js
$i18n.locale = 'zh' // en
window.$it = (str) => {
	return $i18n.t(str)
}
console.log($it('com.btn')); // 按钮
```
