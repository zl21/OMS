## Fkdialog

### 一、组件说明

#### 1. 概述

弹框多选组件，基于普通多选组件，扩展了多条件筛选、反选功能，实现了复杂筛选弹窗功能。

#### 2. 应用场景

- 数据量大且多条件筛选
- 实现反选功能

### 二、配置项说明

```js
props: {
  appendToBody: false;
  canChinese: true;
  fkshow: true;
  isObject: true;
  isOneData: false;
  modalAppendToBody: true;
  rightList: Object(empty);
  tableid: 23281;
  tablename: 'PS_C_SKU';
  title: '弹框多选';
  version: '1.4';
}
```

### 三、使用说明

> js

```js
import fkdialog from 'professionalComponents/fkdialog';
export default {
  components: {
    fkdialog,
  },
};
```

> html

```js
<fkdialog
  v-if="fkDialog.dialog && itemdata.reftable !== 'VP_C_VIP_ACC'"
  :tablename="itemdata.reftable"
  :tableid="itemdata.reftableid"
  :right-list="fkDialog.lists"
  :is-object="itemdata.isObject"
  :append-to-body="layer"
  :fkshow.sync="fkDialog.dialog"
  :is-one-data="itemdata.isOneData"
  :version="itemdata.version"
  @easyData="getFkdialog"
/>
```

### 四、API

#### fkdialog Props

| 属性              | 说明                                                  | 类型          | 默认值   |
| ----------------- | ----------------------------------------------------- | ------------- | -------- |
| version           | 版本号，兼容老接口返回的数据结构，可选值为`1.3`,`1.4` | String        | -        |
| title             | 标题                                                  | String        | 弹框多选 |
| tablename         | 表名                                                  | String        | -        |
| tableid           | 表 id                                                 | Number/String | -        |
| rightList         | 右边数据                                              | Object/String | -        |
| fkshow            | 显示                                                  | Boolean       | true     |
| isObject          | 是否是单对象                                          | Boolean       | false    |
| isOneData         | 是否只能选中一条数据                                  | Boolean       | false    |
| canChinese        | 是否可以模糊搜索中文                                  | Boolean       | true     |
| appendToBody      | 是否将弹框放在 body                                   | Boolean       | false    |
| modalAppendToBody | 是否將蒙层放在 body 上                                | Boolean       | true     |

#### fkdialog events

| 事件        | 说明                 | 返回值                                                                                                                                                  |
| ----------- | -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| easyData    | 触发确认按钮后的回调 | eg: { value: '传向后台的数据', total: '条数', lists: { result: '渲染列表', condition: '条件集合' }, nameList: 'name 集合', idArray: '选中结果所有 id' } |
| dialogClose | 关闭弹窗             | -                                                                                                                                                       |
