## MultipleBox

### 一、组件说明

#### 1. 概述

多选框组件，当所有`options`都选中，`全部`也会被勾选，且与选项在同一行。

#### 2. 应用场景

用于一组可选项多项选择，或者单独用于标记切换某种状态。

### 二、使用说明

> 引用

```js
import multipleBox from 'professionalComponents/multipleBox';
```

> 应用

```js
<multipleBox
  :values="values"
  :options="options"
  @changeOptions="checkOrderTypeChange"
/>
```

```js
export default {
  components: {
    multipleBox,
  },
  data() {
    return {
      options: [
        { title: '正常', value: '1' },
        { title: '换货', value: '2' },
        { title: '补发', value: '4' },
        { title: '虚拟', value: '8' },
        { title: '预售', value: '32' },
        { title: '天猫', value: '602' },
      ],
      values: ['1', '2', '4', '8', '32', '602'],
    };
  },
  methods: {
    checkOrderTypeChange(vals) {
      console.log(vals);
    },
  },
};
```

### 三、API

#### multipleBox Props

| 属性    | 说明             | 类型  | 默认值 |
| ------- | ---------------- | ----- | ------ |
| values  | 选中项目的真实值 | Array | -      |
| options | 可选项           | Array | -      |

#### multipleBox events

| 事件          | 说明                                                                         | 返回值 |
| ------------- | ---------------------------------------------------------------------------- | ------ |
| changeOptions | 在选项状态发生改变时触发，返回已选中的数组。通过修改外部的数据改变时不会触发 | values |
